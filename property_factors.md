# Property Valuation Factors Checklist

## Physical Characteristics
**Data Sources:** [Scottish EPC Register](https://www.scottishepcregister.org.uk/) | [Home Reports](https://www.mygov.scot/buying-a-home/home-report)

- [ ] Number of bedrooms
- [ ] Number of bathrooms
- [ ] Property type (detached/semi-detached/terraced/flat)
- [ ] Age of property
- [ ] Construction type (traditional/non-traditional)
- [ ] Overall condition
- [ ] Level of modernisation
- [ ] Building materials used
- [ ] Roof condition
- [ ] Window type (single/double/triple glazing)
- [ ] Heating system type and condition
- [ ] Electrical system condition
- [ ] Plumbing and water supply
- [ ] Extensions or alterations
- [ ] Internal layout and room flow
- [ ] Storage space
- [ ] Fittings and fixtures quality
